Description

Wenomechainsama is a Discord bot I developed to help relive user usability issues when it comes with recieving and viewing videos from social media sites. What Wenomechainsama does is it detects when a user sends a site from these sites that contains a video it will then be prompted to make a request to download the video. Once thats done, it will scrape critical information such as the title and the User who posted the video from the site. Once downloaded both the video and the information the bot scrapes will be uploaded to discord as a reply from the discord user who sent it. This allows not only for viewing the content but it also provides easy use for other users to download the content thanks to discord video player. Compatible with TikTok, Youtube Shorts and Twitter.
